## ðŸŒŸ Features

### Pages
- **Home Page** - Hero banner, company story, and ice cream menu with 6 flavors
- **Register Page** - User registration with comprehensive form validation
- **Order Page** - Ice cream ordering with delivery/pickup options and payment processing
- **Features Page** - Documentation of implemented features

### Form Validation
- **Registration Form:**
  - Username validation
  - Gender selection
  - Date of birth with age verification (13+ years)
  - Password strength meter
  - Password confirmation
  - Email format validation
  - Phone number validation (10 digits)

- **Order Form:**
  - Ice cream selection with quantity
  - Delivery/Pickup option toggle
  - Address validation (delivery & billing)
  - "Same as delivery address" auto-fill
  - Payment method selection
  - Credit card validation (Visa, MasterCard, American Express)
  - Card number length validation by card type
  - Expiry date validation
  - CVV validation

### Design Features
- Responsive design for all screen sizes
- CSS custom properties (variables) for consistent theming
- Curved hero section with shadow effects
- Modern 2-column footer layout
- Font Awesome icons
- Google Fonts integration (Shantell Sans, Poppins, Open Sans)
